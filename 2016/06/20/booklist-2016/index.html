Sure's Booklist 2016 读书笔记2016

Posted on Jun 20 2016, 26th Mon   |   In books   |   #BOOK   |   Visitors
  : 字数总计： 1,710 个字           : 阅读时长： 6 分钟

1. Think Python (How to Think Like a Computer Scientist)

  • Program testing can be used to show the presence of bugs, but never to show their absence!
    程序测试能被用于展现错误的存在，但是从不会显示其不存在！— Edsger W. Dijkstra

　　平实易懂的语言，为我们呈现了Python的简洁之美。

2. 三体(《三体》、《三体Ⅱ·黑暗森林》、《三体Ⅲ·死神永生》) - 刘慈欣

  • 端午回去，家里在盖房子，小雨的房间被分在二楼靠后。小雨说房子建在树上就好了，她住这片叶子，她哥哥住另一片叶子，她爸爸妈妈住一片大叶子。但是现在科技还不发达，好想将来住在树上啊。——老旭当时一脸惊异，小雨难道看过《三体》？？？！！！

　　场面相当宏大，黑暗森林皆是猎人的推断让人叫绝。时间跨度很长，但最主要的描写还是千禧年前后，也即我们现在生活的时代。小说中涉及到多次关系人类生死存亡的抉择做出，在当时的情况下做出的抉择深刻地反映了人的本性。毁灭人类的不是外在力量，而是人类自身的科学进步与贪婪人性。

3. 底牌 - 阿加莎·克里斯蒂

  • “依我看，从你开始处处小心、处处追求‘安全第一’的时候，就等于已经死了。”
    “完全同意！”露达喊道，“人生就该冒险——如果有机会的话。但总体而言，生活实在太平淡了。”

  • 波洛小声说，“做人就该光明磊落。贵国有位诗人说得好：‘我若不能严守公正，便不能如此爱你。’”
    “是荣誉。”卢克斯摩尔太太微微皱眉纠正。
    “当然——当然——荣誉。‘我若不能严守荣誉……’”

　　虽然之前因为一段英文出自柯南·道尔的《四签名》而有所接触，但最近看的第一部侦探小说则是女作者阿加莎·克里斯蒂的《底牌》。小说比较短，但是涉及到四名解谜者和四名嫌疑人的描写形象鲜明，根据逐步挖掘到的线索进行理论、心理推断的情节引人入胜。作者为女性，却对女侦探、男侦探的思维方式都拿捏的恰到好处，被誉作侦探小说女王，这一点我相信你在两天内看完这篇小说会同意的。作者创作生涯持续了五十余年，幼时几乎没有接受过正规的教育，却到达了柯南·道尔的成就，这一点也是如此的鼓舞人心。

4. JavaScript Python Git 教程 - 廖雪峰

　　看过一遍js基础视频，但是看过文本教程后才发现视频遗漏了太多内容，基础巩固还需看书。

5. 疯狂HTML 5/CSS 3/JavaScript讲义 - 李刚

　　学以致用。工作中Web界面用到了HTML、DOM和Canvas，而Canvas只是稀里糊涂的掌握了控件定位和操作并未更多了解，现在可以详细了解一下。另外，发现HTML5和HTML是相当于WIN10和WIN7的关系。
　　CSS和JS巩固二轮。

6. Core Java (Java核心技术) - Cay S. Horstmann

　　身为Java程序猿中的一员，9.24.2015 - 2016已经工作了一年，再看此书，希望可以解除一些基本概念的疑惑。
　　看书继续ING…

7. 白夜行 - 东野圭吾

　　一个星期的工作之余，看完了第一本东野圭吾的小说，Kindle电子版。读至小说的后半程，逐渐明晰第一章的命案凶手是谁，但读至最后一句“她一次都没有回头”不禁对西本雪穗和桐原亮司两人感到可悲。自小至大都背负着某种仇恨，赚钱重回大阪——那是他们小时候的故乡，而正是他们童年时期的家庭环境带给了他们两个小孩子无比巨大的伤害。然后，接二连三的悲剧开始发生，川岛江利子和筱冢（XiaoZhong）一成，本该是很耀眼的两个人，却没能根据自己的感觉走到一起。附带着另外一些无辜人的牺牲，西本雪穗和桐原亮司最终走到真相的尽头，走到人生的尽头。

8. 解忧杂货店 - 东野圭吾

　　一间废弃的杂货店，通过寄信箱收到书信、牛奶箱寄出书信，连接着过去与现在，现在与未来——这便是消烦解忧的浪矢杂货店。不得不说，作者东野圭吾叙述的构思很巧妙，循着前世与今生是互相影响的主线，将几个看似无关的小故事关联起来。向我们讲述了人生的道理：那些浪矢先生的回信之所以有帮助，是因为寄信人他们本身就有拥有正确的心态，如果他们没有想要认真生活、努力生活的态度，无论别人回答什么，恐怕都帮不了他们。最后，身为男子汉，我们要遵守男人与男人之间的约定。
　　过去、现在和未来依次影响，一封信改变了一生，是追寻内心的指引。一间小小的杂货店，是如此温暖。

9. 嫌疑人x的献身 - 东野圭吾

　　看到小说结尾，才转回开头看案发时间，结果没有找到三月九号，不得不佩服作者的写法精妙。数学传奇石神移花接木，是齿轮在报恩，是爱的深沉。

　　小说中提到了关于P≠NP的两个有趣儿的数学题：

>设计别人解不开的问题和解开那个问题，何者更难？
>自己想出答案和判断别人的答案是否正确，何者较容易？

　　百密也有一疏，唯有事先就做好献身的准备。

10. 学习正则表达式 - Michael Fitzgerald

　　“equals”, “contains”, “find”, “match”, “!”, “if”, “while”… 这些的所有怎么判断一个字符串是否是合法的邮件地址呢？且看正则表达式简而言之。

字符描述：
^：匹配输入的开始位置。
\：将下一个字符标记为特殊字符或字面值。
*：匹配前一个字符零次或几次。
+：匹配前一个字符一次或多次。
(pattern)：与模式匹配并记住匹配。
x|y：匹配 x 或 y。
[a-z]：表示某个范围内的字符。与指定区间内的任何字符匹配。
\w：与任何单词字符匹配，包括下划线。
{n,m}：最少匹配 n 次且最多匹配 m 次
$：匹配输入的结尾。

　　科班出身与中途出家的区别，基本功需要夯实。
